# Java Projects

A collection of Java backend projects demonstrating Spring Boot microservices, cloud integrations, and third-party API integrations.

## Projects

### Java Cloud Service
A Spring Boot microservice designed for cloud deployment with REST API endpoints.

### Third-Party Integration Service
A Spring Boot service demonstrating integration patterns with external APIs and services.

## Tech Stack

- **Language:** Java
- **Framework:** Spring Boot
- **Build:** Maven
- **Architecture:** Microservices, REST APIs

## Getting Started

### Prerequisites
- Java 17+
- Maven

### Running
```bash
cd java-cloud-service
mvn spring-boot:run
```

```bash
cd thirdparty-integration-service
mvn spring-boot:run
```
